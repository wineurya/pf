# Avance — work snapshot

**Avance** is a coach-shaped mobile experience: users state a clear goal, set a timeline and weekly check-ins, and see a plan summary with supportive copy. The UI pairs warm neutrals (beige, deep brown) with a calm, product-led tone and rounded, iOS-like surfaces.

- **Role:** product design exploration (case study route may follow).
- **Visual:** two-screen flow — goal input + plan / nuggets (goal, deadline, check-in, coach quote).
- **Hero image:** Figma `200:109` in `200:88` — canvas `#E3DBD1`; mock 1410.14×1490.18 from artboard. Files: `avance-hero.webp` (≤2560px, q~92) + `avance-hero.png` (≤2000px) for `srcset`; re-export Figma full PNG, then re-run `sharp-cli` if you need to refresh.
